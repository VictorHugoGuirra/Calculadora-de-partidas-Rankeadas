<h1 align="center">Calculadora de Partidas Rankeadas</h1>

<p align="center">
Projeto em JavaScript que calcula o saldo de vitórias de um jogador e determina seu nível ranqueado com base na quantidade de vitórias. O programa recebe os valores de vitórias e derrotas diretamente pelo terminal, processa os dados usando funções e estruturas de decisão, e exibe o resultado final de forma dinâmica.
</p>
